GoFish-
   (1) Download and unzip the zipped file
   (2) Open linux, compile the program using the makefile by typing
       "make" into the linux prompt.
   (3) To run, you will need a .txt file (example.txt in this case). So
       you will type into the linux prompt "./gofish example.txt"
   (4) The output of the GoFish game will be present in the .txt file 
       (example.txt in this case)
